# Tasks App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app) and uses TypeScript.

---

## Technologies Used

- **React** (with TypeScript) for building the UI
- **Tailwind CSS** for styling
- **dnd-kit** for drag and drop functionality
- **Formik** for form state management
- **Yup** for form validation
- **line-clamp** utility from Tailwind CSS for truncating text

---

## Getting Started

### Prerequisites

Make sure you have **Node.js** and **npm** installed on your machine.

---

### Running Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/itskhashman/-UI-Tasks-page.git
2. Navigate to the project directory:

   ```bash
   cd -UI-Tasks-page
4. Install dependencies:
5. 
   ```bash
   npm install
6. Start the development server:
   
   ```bash
   npm start

8. Open your browser and visit:
   
   ```bash
   http://localhost:3000
